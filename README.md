# Scraper-Emag
Am dezvoltat un sistem automatizat de web scraping pentru monitorizarea fluctuațiilor de preț pe platforme de e-commerce.

--Tehnologii: Python, BeautifulSoup4, Requests, JSON, Discord Webhooks.


--Funcționalități cheie:

Implementarea persistentei datelor folosind JSON pentru a compara prețurile între sesiuni.

Integrarea de Discord Webhooks pentru alerte în timp real, cu formatare vizuală (color-coding) în funcție de evoluția prețului.

Optimizarea cererilor HTTP prin User-Agent rotation și randomized delays pentru a evita detectarea de către sistemele anti-bot.
